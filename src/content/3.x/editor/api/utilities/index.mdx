Tiptap Utilities are complementing the Editor API, providing tools that improve and extend your interactions with the editor and content.

## All utilities

| Utility Name                                                 | Description                                               |
| ------------------------------------------------------------ | --------------------------------------------------------- |
| [`HTML Utility`](/3.x/editor/api/utilities/html)             | Handles JSON and HTML transformations server-side.        |
| [`Suggestion Utility`](/3.x/editor/api/utilities/suggestion) | Adds customizable autocomplete suggestions to the editor. |
| [`Tiptap for PHP`](/3.x/editor/api/utilities/tiptap-for-php) | Integrates Tiptap functionalities into PHP projects.      |
