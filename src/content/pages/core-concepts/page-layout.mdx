You can adjust some aspects of the page layout to better fit your needs. The main options are `pageHeaderHeight` and `pageGap`.

## Header and Footer height
- Controls how tall the header and footer area is at the top of each page.
- Default: `50` (pixels)

```js
Pages.configure({
  pageHeaderHeight: 80, // Makes the header and footer area taller
})
```

## Page gap
- Sets the space between each page.
- Default: `50` (pixels)

```js
Pages.configure({
  pageGap: 30, // Reduces the space between pages
})
```

<Callout title="Note">
  Page margins (space around the content) are fixed for each format and cannot be changed yet.
</Callout>
