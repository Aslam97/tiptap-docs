Headers and footers let you add useful information to the top and bottom of every page, like page numbers, document titles, or custom text.

<CodeDemo isPro src="https://deploy-preview-369--tiptap-pro.netlify.app/preview/extensions/pagesheadersfooters" />

## Header and footer positions
- **Left**: Content aligned to the left edge
- **Center**: Content centered
- **Right**: Content aligned to the right edge

You can set each position separately for both headers and footers.

## String templates with tokens
You can use a string for any header or footer. The following tokens are supported:
- `{page}`: Current page number
- `{total}`: Total number of pages

Example:
```js
Pages.configure({
  headerLeft: 'My Project',
  footerRight: 'Page {page} of {total}',
})
```

## Dynamic content (functions)
You can use a function to generate content based on the current page and total pages:

```js
Pages.configure({
  footerRight: (page, total) => `Page ${page} of ${total}`,
  headerCenter: (page) => `Section ${page}`,
  footerLeft: (page) => page === 1 ? 'Confidential' : '',
})
```

## All positions available
You can set left, center, and right for both header and footer:

```js
Pages.configure({
  headerLeft: 'Project X',
  headerCenter: (page, total) => `Section 1`,
  headerRight: (page, total) => `Page ${page} of ${total}`,
  footerLeft: '',
  footerCenter: '',
  footerRight: 'Page {page}',
})
```

## Best practices
<Callout title="Tip">
  Use headers for document titles or section names, and footers for page numbers or legal notes. Keep them short for best results.
</Callout>
