The Notion Editor Template is a fully working setup for the Tiptap editor that recreates the familiar Notion editing experience. It includes collaboration features, AI assistance, emoji support, and advanced formatting options, and ready to customize.

<CodeDemo isLarge src="https://template.tiptap.dev/preview/templates/notion" />

## Installation

Add the Notion Editor Template to your project using the Tiptap CLI:

```bash
npx @tiptap/cli add notion-editor
```

## Usage

After installation, use the `NotionEditor` component in your React or Next.js project:

```jsx
import { NotionEditor } from '@/components/tiptap-templates/notion/notion-editor'

export default function App() {
  return <NotionEditor room="my-document-room" placeholder="Start writing..." />
}
```

## Features

A fully responsive Notion-style rich text editor with real-time collaboration and AI assistance. All components are open source and easy to extend.

- **Real-time collaboration**: Multi-user editing with live cursors and presence indicators
- **AI assistance**: Built-in AI menu for content generation and improvement
- **Responsive design**: Mobile-friendly with adaptive toolbars
- **Dark and light mode**: Supported out-of-the-box
- **Slash commands**: Type `/` to access blocks and formatting
- **Floating toolbar**: Selection-based formatting options
- **Drag and drop**: Reorder blocks with intuitive drag handles
- **Emoji support**: Rich emoji picker with GitHub emoji set
- **Mentions**: @-mention functionality for user references
- **Advanced formatting**:
  - Bold, Italic, Underline, Strikethrough
  - Color and highlight options
  - Superscript and Subscript
  - Code blocks with syntax highlighting
- **Block elements**:
  - Multiple heading levels
  - Lists (bullet, ordered, task lists)
  - Blockquotes
  - Horizontal rules
  - Mathematical expressions
- **Media support**: Image upload with drag-and-drop
- **Link editing**: Rich link management with previews
- **Text alignment**: Left, Center, Right, Justified
- **Undo / Redo**: Complete history management
- **Context menus**: Right-click options for enhanced productivity

## Collaboration Setup

The Notion template includes built-in collaboration features. To use them:

1. **Room-based collaboration**: Each document needs a unique room identifier
2. **User presence**: Automatic user avatars and live cursors
3. **Conflict resolution**: Operational transformation handles concurrent edits
4. **Persistence**: Documents are automatically saved and synced

```jsx
// Example with custom room and user settings
<NotionEditor room="project-meeting-notes" placeholder="Share your thoughts..." />
```
