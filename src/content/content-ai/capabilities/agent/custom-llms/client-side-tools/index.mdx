Client-side tools are tools that run in the browser. Use them to interact with the editor's content. For example:

- A tool that counts the number of words in the document
- A tool that replaces all instances of a word in the document
- A tool that formats the document.

The guide has different versions depending on the AI library you use. We currently support the following libraries:

- [Vercel AI SDK](/content-ai/capabilities/agent/custom-llms/client-side-tools/vercel-ai-sdk)
- [OpenAI Chat Completions API](/content-ai/capabilities/agent/custom-llms/client-side-tools/openai-chat-completions)
- [OpenAI Responses API](/content-ai/capabilities/agent/custom-llms/client-side-tools/openai-responses)
- [Anthropic Claude Messages API](/content-ai/capabilities/agent/custom-llms/client-side-tools/anthropic-messages)
